About the Work

'Peel-off'는 탈피라는 생물학적 현상을 디지털 환경에서 재해석한 인터랙티브 미디어 아트입니다. 사용자는 실물 카드를 매개로 가상의 공간에서 탈피를 체험하며, 이 과정을 반복함으로써 시간의 흐름과 변화의 순간을 관찰합니다.

Technical Tools

AI & Data: Gemini Teachable Machine

3D & Graphic: Three.js Tripo 3D

Development: HTML5 CSS3 p5.js

How to Use
  • 1. 화살표(<<, >>)를 클릭하여 이미지를 선택 후 Select 버튼을 누릅니다.
  • 2. 선택한 이미지에 해당하는 그림 카드를 화면에 비춥니다.
  • 3. 3D 이미지가 나오면 Peel-off 버튼을 누릅니다.
  • 4. 탈피가 끝나면 마우스로 돌려보며 새로워진 모습을 관찰합니다.
  • 5. 탈피를 반복하여 표피가 쌓여가는 것을 관찰합니다.
